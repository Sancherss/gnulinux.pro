# Файлы и директории

## Вопросы и ответы (по gnulinux.pro)

### 1. С помощью какой программы можно работать с файлами в графическом интерфейсе?

**Ответ:** с помощью файлового менеджера.

> На практике: Nautilus (GNOME), Dolphin (KDE), Thunar (XFCE).

---

### 2. Как узнать текущую директорию в командной строке?

**Ответ:** с помощью команды `pwd`.

```bash
pwd
# /home/user/gnulinux-pro-internship
```

---

### 3. Что удобнее — полный или относительный путь?

**Ответ:**  
- **Полный путь** — от корня `/` до текущей директории.  
- **Относительный путь** — от текущей директории до нужной.  

**Удобство зависит от задачи.**

| Ситуация | Какой путь лучше |
|----------|------------------|
| Быстро ходить по папкам в терминале | относительный |
| Писать скрипты или конфиги | полный |

---

### 4. Как сделать, чтобы не писать команду или путь целиком?

**Ответ:** использовать клавишу **Tab** (автодополнение).  

- Один Tab — дополняет, если вариант один.  
- Два Tab — показывает все возможные варианты.

```bash
cd /us[Tab]       → /usr/
ls Doc[Tab][Tab]  → Documents/ Downloads/
```

---

### 5. Как вернуться в домашнюю директорию?

**Ответ:**  
- `cd ~`  
- или полный путь `/home/имя_пользователя`

> Также работает просто `cd` (без аргументов) — это самый короткий способ.

## Задание 12
**Из `/tmp` удалить `~/testdir1` и `testdir2/testdir22` с выводом информации об удалении.**

```bash

```

---

## Итог по практике

- `mkdir -p` — создание вложенных папок одной командой  
- `rmdir` — безопасное удаление пустых папок  
- `rm -r` — рекурсивное удаление  
- `ls -R` — рекурсивный просмотр  
- `cd -` — прыжок между двумя папками  
- Кавычки и `\` — работа с пробелами и спецсимволами  
- `~` — сокращение для домашней директории
