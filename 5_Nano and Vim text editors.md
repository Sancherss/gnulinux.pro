# Работа с текстовыми редакторами: nano и vi


## Подготовка

```bash
mkdir myfiles                    # создаём директорию
cp /etc/passwd ~/myfiles/        # копируем файл для работы
cd myfiles/                      # переходим в неё
```

## Редактор nano

### Задание 1: Переход к строке и удаление

**Как я это делаю:**

```bash
nano passwd

Ctrl + U          # поиск
Ctrl + T          # режим "Go to line/column"
16 → Enter        # ввожу номер строки

Ctrl + Del        # удаляю всю строку

Ctrl + O → Enter  # сохранить
Ctrl + X          # выйти
```

### Задание 2: Замена false на true (ручной способ)

```bash
Ctrl + W          # поиск
false → Enter     # найти слово
Backspace/Del     # удалить
true              # написать новое

# Повторить для всех вхождений

Ctrl + O → Enter  # сохранить
Ctrl + X          # выйти
```

> **Альтернатива (быстрая замена):** `Ctrl + \` → `false` → `true` → `A` (заменить всё)

### Задание 3: Постоянная нумерация строк

```bash
# Создаю файл конфигурации
touch ~/.nanorc

# Открываю его
nano ~/.nanorc

# Добавляю строку
set linenumbers

# Сохраняю и выхожу
Ctrl + O → Enter
Ctrl + X
```

**Теперь нумерация строк включается автоматически при каждом запуске nano.**

#### Другие полезные настройки `~/.nanorc`

```bash
set tabsize 4           # размер табуляции
set autoindent          # автоотступ
set mouse               # поддержка мыши
set backup              # создавать резервные копии
```

## Редактор vi (vim)

### Основные режимы

| Режим | Как войти | Как выйти |
|-------|-----------|-----------|
| **Нормальный** (Normal) | `Esc` | — |
| **Ввода** (Insert) | `i`, `a`, `o` | `Esc` |
| **Командный** (Cmd-line) | `:` | `Esc` или `Enter` |

### Задание 4: Вставить строку между 5 и 6

```bash
vi passwd

Esc              # нормальный режим (на всякий случай)
5G               # перейти на 5 строку
o                # создать строку ниже (режим ввода)
Hello            # написать текст
Esc              # выйти в нормальный режим
:wq              # сохранить и выйти
```

### Задание 5: Добавить строку в конец и выйти без сохранения

```bash
vi passwd

GG               # перейти в конец файла (Shift + G)
o                # новая строка ниже
World            # написать текст
Esc              # выйти в нормальный режим
:q!              # выйти без сохранения
```

> `:wq` — сохранить и выйти  
> `:q!` — выйти без сохранения (принудительно)

## Проверка результата

### Задание 6: Поиск строки с Hello

```bash
cat passwd | grep -n Hello
```

**Вывод:**
```
6:Hello
```

### Полезные команды `grep`

| Команда | Что делает |
|---------|-------------|
| `grep -n Hello passwd` | найти строку с номером |
| `grep -ni hello passwd` | искать без учёта регистра |
| `grep -n ^root passwd` | строки, начинающиеся с root |
| `grep -n false$ passwd` | строки, заканчивающиеся на false |

### Nano

| Действие | Команда |
|----------|---------|
| Поиск | `Ctrl + W` |
| Перейти на строку | `Ctrl + T` (после поиска) |
| Удалить строку | `Ctrl + Del` |
| Сохранить | `Ctrl + O` |
| Выйти | `Ctrl + X` |

### Vim

| Действие | Команда |
|----------|---------|
| На строку 5 | `5G` или `:5` |
| В конец файла | `GG` |
| Новая строка снизу | `o` |
| Удалить строку | `dd` |
| Сохранить и выйти | `:wq` |
| Выйти без сохранения | `:q!` |
